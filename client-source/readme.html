ATTENTION!

The game is still being in development so it may contain bugs and errors!

If you find any bugs or errors, please contact me and tell about them. I know some of them and I will try to fix them in the closest future, but for now I leave here a list of known errors and their solutions:

-> Sometimes ball may got stuck in the celling - move the paddle to fix it. -> Sometimes controls may behave weird after few multiplayer games - reload the page to fix it. -> Sometimes you may have troubles with connection to server - reload the page to fix it.

I`ve made this game as mine diploma for college in May, 2020. I`ve been inspired by COVID-19 pandemic because a lot of people, including me had to work from home but had a little bits of time between different work calls and etc. So I have decided to make some kind of simple, relaxing and entertaining game, which wouldn`t be demanding to computer resources, so you can keep it working in browser tab.
So please enjoy this game, hope you will like it.

Also contact me and give me more ideas for splash text in the main menu, if you have such ;)
