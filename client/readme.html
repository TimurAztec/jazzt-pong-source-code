ATTENTION!

The game is still being in development so it may contain bugs and errors!

If you found any bugs or errors, please contact me and tell about them. I know some of them and I will try to fix them in closest future, but for now I leave here a list of known bugs and their solutions:

-> Sometimes 2 players may control 1 paddle - to fix it reload the game page. -> Sometimes ball may got stuck between paddles and walls - to fix it move the paddle.

I`ve made this game as mine diploma for college in May of 2020. I`ve been inspired by COVID-19 pandemic because a lot of people, me including had to work from home but had a little bits of time between different work calls and etc. So I have decided to make some kind of simple, relaxing and entertaining game, which wouldn`t be demanding to computer resources, so you could keep it working in browser tab.
So please enjoy this game, hope you will like it.

Also contact me and give me more ideas for splash text in main menu, if you have such ;)
